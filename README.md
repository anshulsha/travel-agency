# travel-agency
## A full stack project with backend, frontend, nginx, db

## Getting Started

### Prerequisites

- Node.js (>=12.x)
- npm (>=6.x)

### Installation

#### Docker is required to run this project and it should be up and running

1. Clone the repository:

```bash

git clone https://github.com/anshulsha/travel-agency.git

sh setup.sh


```
